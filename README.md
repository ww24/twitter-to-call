Twitter to Call
===============

Twitter を使って電話をかける Web アプリケーションです。

Requirement
-----------
* [Node.js](http://nodejs.jp/nodejs.org_ja/docs/v0.10/) (v0.10)
* [Redis](http://redis.io/download) (v2.6)
* [Twilio](http://twilio.kddi-web.com/) Account

How to Setup & Start/Stop
-------------------
### Setup
`npm install`

### Start
`npm start`

### Stop
`npm stop`
